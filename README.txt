This website is the first version of Chayn Belgium, the belgian chapter of Chayn International. CHAYN is a global volunteer network addressing gender-based violence by creating intersectional survivor-led resources online. We started in 2013, and have since reached more than 300 000 people and 1.2 million views across the world.
